Documentation Notes

SkylinkCC is a implementation from Skylink to create a control center like use-case. Things to take note are:

  • Skyway.init() is required to be called before connect()
  • To disconnect users from the lobby or the room, call Skyway.leaveRoom() instead.
  • If there's no lobbyRoom specified, 'MAIN' is the main lobby
  • Check out the rest of the other functionalities in the Skyway documentation.

  • ROOM_LOCKED {String}

    Room is locked and user is locked from joining the room.

  • ROOM_FULL {String}

    Persistent meeting. Room is full.

  • DUPLICATED_LOGIN {String}

    User has same id

  • SERVER_ERROR {String}

    Server has an error

  • VERIFICATION {String}

    Verification for roomID

  • EXPIRED {String}

    Persistent meeting. Room has expired and user is unable to join the room.

  • ROOM_CLOSED {String}

    Persistent meeting. Room has expired and is closed, user to leave the room.

  • ROOM_CLOSING {String}

    Persistent meeting. Room is closing soon.

  • OVER_SEAT_LIMIT {String}

    Seat limit is hit. API Key do not have sufficient seats to continue.
